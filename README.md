# Платформа торговой сети электроники

## 📋 ОПИСАНИЕ ПРОЕКТА
Веб-приложение с API-интерфейсом и админ-панелью для управления сетью по продаже электроники.
Реализована иерархическая структура сети из трех уровней: завод, розничная сеть, индивидуальный предприниматель.

## 🚀 ТЕХНИЧЕСКИЕ ТРЕБОВАНИЯ
- ✅ Python 3.8+ (использован Python 3.13.3)
- ✅ Django 3+ (использован Django 4.2.7)
- ✅ DRF 3.10+ (использован DRF 3.14.0)
- ✅ PostgreSQL 10+ (настроен в settings.py, для разработки используется SQLite)

## 📦 ФУНКЦИОНАЛЬНЫЕ ТРЕБОВАНИЯ

### 1. МОДЕЛЬ СЕТИ (иерархическая структура)
Реализованы три уровня иерархии:
- **Уровень 0**: Завод
- **Уровень 1**: Розничная сеть
- **Уровень 2**: Индивидуальный предприниматель

Каждое звено сети ссылается только на одного поставщика оборудования.

### 2. ЭЛЕМЕНТЫ ЗВЕНА СЕТИ
| Элемент | Реализация |
|---------|------------|
| **Название** | `name = models.CharField()` |
| **Контакты** | Отдельная модель `Contact` |
| - email | `email = models.EmailField()` |
| - страна | `country = models.CharField()` |
| - город | `city = models.CharField()` |
| - улица | `street = models.CharField()` |
| - номер дома | `house_number = models.CharField()` |
| **Продукты** | Отдельная модель `Product` |
| - название | `name = models.CharField()` |
| - модель | `model = models.CharField()` |
| - дата выхода | `release_date = models.DateField()` |
| **Поставщик** | `supplier = models.ForeignKey('self')` |
| **Задолженность** | `debt = models.DecimalField(max_digits=10, decimal_places=2)` |
| **Время создания** | `created_at = models.DateTimeField(auto_now_add=True)` |

### 3. АДМИН-ПАНЕЛЬ
- ✅ Вывод всех созданных объектов
- ✅ **Ссылка на поставщика** (кликабельная в списке объектов)
- ✅ **Фильтр по названию города** (в правой панели фильтрации)
- ✅ **Admin action** для очистки задолженности у выбранных объектов

### 4. API (Django REST Framework)
- ✅ **CRUD для модели поставщика** (все операции)
- ✅ **Запрет обновления поля "Задолженность"** через API (`read_only_fields`)
- ✅ **Фильтрация объектов по стране** (`?country=Россия`)
- ✅ **Права доступа** только для активных сотрудников
